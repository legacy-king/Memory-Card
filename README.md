# Memory Card Game

A memory card matching game built with React. Click on cards without clicking the same one twice — your score increases with each unique click, and resets if you click a repeat. Cards reshuffle after every successful click to keep you on your toes.

## Features

- Dynamic card grid fetched live from the PokeAPI
- Score and Best Score tracking across attempts
- Cards reshuffle on every successful click
- Game resets on a repeat click, while Best Score persists
- Responsive grid layout with hover effects

## Built With

- React
- Vite
- PokeAPI
- CSS

## Getting Started

\`\`\`bash
npm install
npm run dev
